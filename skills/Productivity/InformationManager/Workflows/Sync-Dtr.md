# Sync-Dtr Workflow

Sync DTR (Down The Rabbit Hole) metrics into Kaya's context.

**Category:** Sync (Source → Kaya)
**Trigger:** `sync dtr`, `refresh dtr`, `update dtr context`, `/info dtr`

## Purpose

Pull current metrics FROM DTR Google Sheets INTO `~/.claude/context/DtrContext.md`.

This is a **Sync** workflow: it pulls data from Google Sheets into Kaya's context directory.

## Execution Steps

### 1. Fetch Data via CLI

```bash
# Goal achievement
kaya-cli sheets get 1iiX2qfRn6Gx1q1yLu5Xu8nV-x7S_KA45gykjgQKTJdw --range "A1:Z20" 2>/dev/null

# Habit building
kaya-cli sheets get 1xrGAGvKlgckHbjnMevs9ZhlwtNWaUL5CzzgRQ82X9LA --range "A1:AM10" 2>/dev/null
```

### 2. Parse Response

Extract:
- Current WIG scores
- Lead measure performance
- Habit consistency rates
- Alignment metrics

### 3. Write Context File

Write to `~/.claude/context/DtrContext.md`:

```markdown
---
tags: [context, ai-context, dtr]
last_updated: {{TIMESTAMP}}
gathered_by: InformationManager
entries_count: {{METRIC_COUNT}}
---

# DTR Metrics Context

## Summary
Current metrics from DTR tracking system.

## WIG Progress (Q1 2026)

| WIG | Current | Target | Gap |
|-----|---------|--------|-----|
| G0 (Media) | 5.3 hrs | 3.0 hrs | -2.3 |
| G1 (Friends) | 0 | 2 | -2 |
| G2 (Alignment) | 1.8 | 3.0 | -1.2 |

## Lead Measures (7-Day Rolling)

| Measure | Rate | Target |
|---------|------|--------|
| S0 (Boredom Blocks) | 64.3% | 100% |
```

### 4. Return Summary

- Key WIG status
- Any errors encountered
