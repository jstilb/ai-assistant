# Sync-Learnings Workflow

Sync learnings context from the Memory system into Kaya's context.

**Category:** Sync (Source ‚Üí Kaya)
**Trigger:** `sync learnings`, `refresh learnings`, `update learnings context`, `/info learnings`

## Purpose

Aggregate recent learnings and patterns FROM `MEMORY/LEARNING/` INTO `~/.claude/context/LearningsContext.md`.

This is a **Sync** workflow: it pulls data from the memory system into Kaya's context directory.

## Domain Configuration

| Source | Path |
|--------|------|
| Ratings | `MEMORY/LEARNING/SIGNALS/ratings.jsonl` |
| Synthesis | `MEMORY/LEARNING/SYNTHESIS/` |
| Algorithm | `MEMORY/LEARNING/ALGORITHM/` |

## Execution Steps

### 1. Read Ratings File

```bash
tail -50 ~/.claude/MEMORY/LEARNING/SIGNALS/ratings.jsonl
```

Parse JSON lines to extract:
- Recent ratings (last 20-50)
- Average score
- Rating trends

### 2. Scan Synthesis Directory

List recent synthesis files:
```bash
ls -lt ~/.claude/MEMORY/LEARNING/SYNTHESIS/*/*.md | head -10
```

### 3. Scan Algorithm Learnings

List recent algorithm learnings:
```bash
ls -lt ~/.claude/MEMORY/LEARNING/ALGORITHM/*/*.md | head -10
```

### 4. Write Context File

Write to `~/.claude/context/LearningsContext.md`:

```markdown
---
tags: [context, ai-context, learnings]
last_updated: {{TIMESTAMP}}
gathered_by: InformationManager
entries_count: {{PATTERN_COUNT}}
---

# Learnings Context

## Summary
{{PATTERN_COUNT}} patterns and learnings captured.

## Recent Ratings

- **Average Score (Last 20):** {{AVG}}
- **Total Ratings:** {{COUNT}}
- **Trend:** {{UP/DOWN/STABLE}}

## Recent Synthesis

### {{MONTH}}
- synthesis-file-1.md
- synthesis-file-2.md

## Algorithm Learnings (Recent)

- 2026-02-03: Topic of learning
- 2026-02-02: Another learning

## Key Patterns

{{Extracted patterns from synthesis files}}

---
*Generated by InformationManager/Workflows/Sync-Learnings.md*
```

### 5. Return Summary

Output:
- Total patterns found
- Average rating
- Any errors encountered
