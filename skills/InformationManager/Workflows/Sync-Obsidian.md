# Sync-Obsidian Workflow

Sync Obsidian vault context into Kaya's context.

**Category:** Sync (Source → Kaya)
**Trigger:** `sync obsidian`, `refresh obsidian`, `update obsidian context`, `/info obsidian`

## Purpose

Scan Obsidian vault structure and write summary TO `~/.claude/context/ObsidianContext.md`.

This is a **Sync** workflow: it pulls a summary of the vault into Kaya's context directory.

**Note:** For updating context files IN the vault itself, use `Refresh-Vault` or `Refresh-VaultFolder`.

## Domain Configuration

| Setting | Value |
|---------|-------|
| **Vault Location** | `~/obsidian/` |
| **Output** | `~/.claude/context/ObsidianContext.md` |

## Execution Steps

### 1. Wraps Refresh-Vault

This workflow can invoke the more comprehensive `Refresh-Vault` workflow to update vault-internal context first.

```
-> Invoke Refresh-Vault workflow (optional, if context needs refreshing)
```

### 2. Alternative: Direct Scan

If just syncing context to Kaya (not refreshing vault):

```bash
# Count folders
ls -d ~/Desktop/obsidian/*/ 2>/dev/null | wc -l

# Count notes per folder
for dir in ~/Desktop/obsidian/*/; do
  count=$(ls "$dir"*.md 2>/dev/null | wc -l)
  echo "$(basename "$dir"): $count"
done
```

### 3. Write Context File

Write to `~/.claude/context/ObsidianContext.md`:

```markdown
---
tags: [context, ai-context, obsidian]
last_updated: {{TIMESTAMP}}
gathered_by: InformationManager
entries_count: {{FOLDER_COUNT}}
---

# Obsidian Vault Context

## Summary
{{TOTAL_NOTES}} notes across {{FOLDER_COUNT}} folders.

## Vault Structure

**Location:** `~/obsidian/`

| Folder | Notes | Last Modified |
|--------|-------|---------------|
| Writing | 45 | 2026-02-01 |
| Projects | 23 | 2026-02-03 |
...

## Key MOCs (Maps of Content)

- `Index.md` - Main vault index
- `Writing/_Index.md` - Writing projects
...

## Recent Activity

Notes modified in last 7 days:
- note1.md (Writing)
- note2.md (Projects)

---
*Generated by InformationManager/Workflows/Sync-Obsidian.md*
```

### 4. Return Summary

Output:
- Total notes found
- Folder count
- Recent activity count
- Any errors encountered
