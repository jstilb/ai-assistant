# Development

Software development tools — covering skill creation scaffolding, CLI tool generation, UI component building, browser automation, canvas project management, and Unix CLI assistance.

## Sub-Skills

| Sub-Skill | Triggers | Load |
|-----------|----------|------|
| **CreateSkill** | create skill, new skill, scaffold skill, skill template | `Development/CreateSkill/SKILL.md` |
| **CreateCLI** | create CLI, new CLI, CLI tool, command line tool | `Development/CreateCLI/SKILL.md` |
| **UIBuilder** | UI builder, build UI, component, interface design | `Development/UIBuilder/SKILL.md` |
| **Browser** | browser, web browser, browser automation, web automation | `Development/Browser/SKILL.md` |
| **Canvas** | canvas, canvas project, portfolio canvas | `Development/Canvas/SKILL.md` |
| **UnixCLI** | unix, CLI, shell, bash, terminal, command line | `Development/UnixCLI/SKILL.md` |

## Workflow Routing

When a sub-skill is identified from the table above:
1. Read the sub-skill's SKILL.md for its full routing and workflow list
2. Execute the appropriate workflow from that sub-skill's directory
