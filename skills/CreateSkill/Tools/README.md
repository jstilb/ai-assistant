# CreateSkill Tools

This is a documentation/SOP skill. It does not contain executable tools.

The Tools/ directory exists to satisfy the canonical skill structure checklist, which requires every skill to have a Tools/ directory. For skills that are purely documentation-driven (like CreateSkill), this directory remains empty aside from this README.

If automation is added in the future (e.g., a skill validator script), those tools would be placed here following TitleCase naming: `ValidateStructure.ts`, `ValidateStructure.help.md`.
